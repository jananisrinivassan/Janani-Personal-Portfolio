Number Guess Game

Les utilisateurs sont invités à entrer leurs suppositions et le jeu leur fournit des indices tels que "trop haut" ou "trop bas" pour les guider vers la bonne réponse. Multiplication Table


Le projet Table de multiplication, développé à l'aide de HTML, CSS et JavaScript, fournit une interface interactive dans laquelle les utilisateurs peuvent saisir un nombre. Une fois le nombre saisi, l'application génère et affiche dynamiquement la table de multiplication correspondant à ce nombre. Cette fonctionnalité permet aux utilisateurs d'explorer et d'apprendre les tables de multiplication de manière interactive sur le web.

Kitchen Connect


Kitchen Connect est une plateforme web innovante conçue avec JavaScript, HTML, CSS et MySQL, destinée aux cuisiniers amateurs passionnés par le partage de leurs créations culinaires. Le projet facilite la création d'un marché communautaire où les cuisiniers peuvent présenter leurs plats et entrer en contact avec des clients locaux intéressés par des repas faits maison. En intégrant des interfaces conviviales et des bases de données MySQL sécurisées, Kitchen Connect permet aux cuisiniers amateurs de présenter leurs spécialités, de gérer les commandes et d'organiser les livraisons de nourriture, créant ainsi un environnement pratique et fiable pour les passionnés de cuisine à domicile.

Contact Form 1

Le projet Contact Form est une application web complète conçue à l'aide de HTML, CSS, PHP et SQL pour permettre aux utilisateurs de soumettre leurs coordonnées de manière transparente. Ce projet met en œuvre un système sécurisé et efficace dans lequel les détails saisis par l'utilisateur, tels que le nom, l'adresse électronique et les messages, sont collectés au moyen d'un formulaire HTML intuitif. Les données sont traitées et stockées en toute sécurité dans une base de données PHPMyAdmin utilisant PHP et SQL, garantissant ainsi un stockage et une gestion fiables des informations soumises par les utilisateurs. Avec une interface conviviale et une intégration robuste du backend, le projet Contact Form offre aux utilisateurs une approche rationalisée pour transmettre leurs données en toute sécurité.

Technologies Used:
Frontend: HTML, CSS
Backend: PHP, SQL
Database: PHPMyAdmin
